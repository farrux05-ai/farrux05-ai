##  Featured Project

### [ B2B SaaS RevOps Pipeline](https://github.com/farrux05-ai/b2b-saas-revops)
End-to-end revenue operations data pipeline for B2B SaaS companies.

**The Problem:**  
Sales asks "Is this account paid?" → checks Stripe manually  
CS asks "Are they using the product?" → checks Mixpanel manually  
Finance asks "What's our MRR?" → calculates in Excel

**The Solution:**  
Unified data pipeline: `HubSpot + Stripe + Mixpanel + Intercom → dbt → 360° account view`

**Impact:**
- ✅ Single source of truth for 10,000+ accounts
- ✅ 167 automated data quality tests
- ✅ Real-time Streamlit dashboards for revenue teams
- ✅ SCD Type 2 snapshots tracking account health changes

**Tech:** dbt · PostgreSQL · Streamlit · Python · SCD Type 2
